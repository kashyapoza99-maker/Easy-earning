# Easy Earning (Demo UI Template)

Black & Gold UI template for a USDT (TRC20) wallet/admin workflow platform.

## Features (UI Demo)
- User pages: Dashboard, Recharge (QR only), Withdraw, KYC, Referral, Support Chat
- Admin pages: Admin Login + Admin Dashboard UI
- Mobile friendly responsive design

## Run Locally
```bash
npm install
npm run dev
```

## Deploy (Render)
- Build command: `npm install && npm run build`
- Start command: `npm run preview`
